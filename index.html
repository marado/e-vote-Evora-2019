e-vote Évora 2019

De que se trata este site?


Em 2019, vai haver um piloto de voto electrónico em Évora, para as eleições europeias.
Este pequeno site pretende reunir informações públicas sobre o piloto.

Referências

Segue-se uma lista de links para sites externos, públicos, com informação relevante sobre o projeto-piloto. Sempre que possível, alojamos aqui também uma versão local dos mesmos, para que se mantenham disponíveis mesmo que haja indisponibilidade dos serviços.
